# SwagLabs Automation Framework

Automated UI testing for SauceDemo SwagLabs application using Selenium WebDriver with Java.

## Tech Stack
- **Language**: Java 17
- **Automation Tool**: Selenium WebDriver 4.15
- **Test Framework**: TestNG
- **Build Tool**: Maven
- **Design Pattern**: Page Object Model

## Features Covered
- ✅ Login functionality validation
- ✅ Data-driven testing approach  
- ✅ Cross-browser support setup
- ✅ TestNG detailed HTML reports
